我构建了一个面向机器人强化学习训练的实验管理 Agent，解决 IsaacLab/RSL-RL 实验中参数组合多、日志分散、失败原因定位慢的问题。核心流程是：Planner Agent 根据任务目标拆解实验变量，如奖励函数、观测维度、动作空间和域随机化范围；Code Agent 自动生成或修改 YAML 配置、训练脚本和评估脚本；Runner Agent 在 Linux 环境中批量启动训练任务并监控显存、loss、success rate；Evaluator Agent 自动读取 TensorBoard 曲线和仿真视频，判断训练是否发散、是否出现 reward hacking，并生成下一轮调参建议。
该系统将一次实验迭代从手动配置、运行、记录的 2-3 小时缩短到约 40 分钟，同时减少了因配置遗漏导致的无效训练。我主要将它用于机械臂抓取策略训练和仿真参数搜索，帮助更快定位奖励函数设计问题，提高实验复现和对比效率。
